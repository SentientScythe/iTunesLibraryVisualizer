To run Music.html you must:
1. Close ALL running instances of Google Chrome
2. Run chrome.exe with the argument --allow-file-access-from-files by(
 a. Adding the argument to the properties of a shortcut to chrome.exe
 -OR-
 b. Running chrome.exe from the command line and passing the argument
)

This allows D3 to access iTunesLibrary.csv directly from the folder.
